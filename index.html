Covered SF

This was my final project at GA's WDI and it was also my favorite. The site is designed to make it easy for anyone in San Francisco to search for doctors and physicians based on what health insurance they accept, focusing mainly on Covered California, our state's new health insurance exchange. It also allows for anyone to view reviews on those doctors and their locations through Google Maps and Yelp's API. In order to actually get this list of doctors, however, I had to make my own API since one didn't exist. I used ruby to make some scripts and manually get the information on each doctor off of their insurance's web directories.

Flightshare

This was my 2nd project at WDI and it was also a group project. The idea was to make it super easy to share flight information with anyone you choose as well as have the site update you automatically via text if your flight was delayed. The site was built using Ruby on Rails, Twilio's API and FlightStats API. I primarily worked on the backend, doing the database, the API calls, as well as the logic for the automated workers for sending emails and texts.

PetHub

This was my first big project at WDI and it shows. Built entirely in Ruby on Rails, it is fairly basic but the idea was interesting to me. Basically it is a pet sharing/social site so people that need their pets watched can loan them to people who want to watch pets for a few hours but are not able to own one.

Ritly
